# CodeAlpha_student_management_system
**Student management system** built in C using **structures** and **files handling** . Supports **add , update , delete , search and display student records** efficiently.
This project is developed as part of the Code Alpha Internship.

## Features

- Add new student records
- Update existing Student details
- Delete Student records 
- Search for a student by ID or name
- Display all Student records in a structured format
- Persistent storage using file handling

---

## Concepts Used 
- C Programming
- Structures
- File Handling
