This project is developed as part of the Code Alpha Internship.
#Features
-Add student
-Update Student
-Delete Student 
-Search student
-Display Student

#Concepts Used 
-C Programming
-Structures
-File Handling

#How to run
-compile using gcc
-Run in terminal
